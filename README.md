Este projeto implementa uma automação completa de WhatsApp utilizando Python, integrada ao Google Sheets para envio de notificações em tempo real.
O sistema monitora continuamente uma planilha, identifica novos registros e envia mensagens automaticamente para contatos ou grupos no WhatsApp, marcando cada item processado como “ENVIADO” para evitar duplicações. 
A solução inclui logging detalhado, variáveis de ambiente para configuração, suporte a modo de teste e fluxo totalmente automatizado, permitindo que empresas recebam alertas imediatos sempre que um novo pedido é criado via Google Forms. 
É uma automação estável, simples de manter e projetada para funcionar como um servidor contínuo, garantindo agilidade, confiabilidade e redução de tarefas manuais.
