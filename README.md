Este projeto implementa uma automação completa de WhatsApp utilizando Python, integrada ao Google Sheets para envio de notificações em tempo real.
O sistema monitora continuamente uma planilha, identifica novos registros e envia mensagens automaticamente para contato(s) no WhatsApp. 
A solução inclui logging detalhado, variáveis de ambiente para configuração, suporte a modo de teste e fluxo totalmente automatizado em que a empresa receberá alertas imediatos sempre que um novo pedido for criado via Google Forms. 
É uma automação estável, simples de manter e projetada para funcionar como um servidor contínuo, confiável, reduz perda de informações e tarefas manuais.
